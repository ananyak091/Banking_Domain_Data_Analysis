# Banking_Domain_Data_Analysis

# üè¶ Banking Analytics Dashboard  
**Power BI | Python (EDA) | SQL (Database)**

This project is a complete Banking Analytics Dashboard designed using **Power BI**, supported by **Python-based Exploratory Data Analysis (EDA)** and an **Excel-based dataset**.  
The dashboard provides a multi-page analytics system similar to a real banking web portal, enabling deep insight into client behavior, loans, deposits, and account distribution.

---

## üìä **Dashboard Overview**

### **Key KPIs**
- **Total Clients:** 3,000  
- **Total Loans:** 1.77 Bn  
- **Total Deposits:** 3.77 Bn  
- **Total Checking Accounts:** 963.28 M  
- **Total Saving Accounts:** 698.73 M  
- **Total Business Lending:** 2.60 Bn  

These KPIs dynamically change with slicers such as **Year of Joining**, **Gender**, **Bank Type**, **BRId**, and more.
